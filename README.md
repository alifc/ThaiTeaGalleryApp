# Thai Tea Gallery App
UTS Pemrograman Mobile ITBS-JM

## Deskripsi Aplikasi
Aplikasi ini menampilkan berbagai varian minuman Thai Tea beserta resep dan gambarnya. Dibuat menggunakan Android Studio dan Java.
